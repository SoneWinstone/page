Linux系统更改MySQL密码

发表于 2017-04-25 | 阅读次数

1. 找到MySQL配置文件my.cnf

MySQL配置文件在Windows下叫my.ini，在MySQL的安装目录下；
在Linux下叫my.cnf，在/etc目录下

2. 修改配置

在[mysqld]的下面加入一行skip-grant-tables，然后按ESC键 :wq保存并退出

3. 重新启动MySQL

4. 修改root密码

5. 改回配置文件

改好之后，再修改一下my.cnf这个文件，把我们刚才加入的skip-grant-tables这行删除或注释，保存退出再重启mysql服务
